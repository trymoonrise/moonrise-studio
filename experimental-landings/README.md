# Experimental Landings (28–67)

40 full-site experimental landing pages that extend the Test Templates series.

## Why this folder?

`testtemplates/` is currently OS-locked (EPERM / Access denied) on this machine, so new templates were written here instead of appending `28–67` under `testtemplates/`.

## Interlinks

- This index ↔ `Website Presets/index.html`
- This index ↔ `testtemplates/index.html`
- Each landing footer links all three galleries

## Regenerate

```bash
node experimental-landings/generate.js
```
