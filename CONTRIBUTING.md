# Contributing to Moonrise Studio

Thanks for your interest in Moonrise.

This project is built for freelancers and creators who help local businesses get online. Useful contributions include product feedback, bug reports, and clear ideas that improve the selling and delivery experience.

## Ways to help

- Report bugs you hit while using Studio
- Suggest features that make finding, building, presenting, or getting paid easier
- Share feedback on clarity, wording, and user flow
- Point out broken links, confusing pages, or accessibility issues

## Before you open an issue

1. Check whether the issue already exists
2. Include what you were trying to do
3. Include what happened instead
4. Add screenshots when they help

## Feature requests

Good feature requests usually answer:

- Who is this for?
- What problem does it solve?
- What would “done” look like?

## Code and product changes

Moonrise Studio is an active product. Not every suggestion can be accepted, but thoughtful feedback is welcome.

If you are proposing a change, keep the focus on the user experience and business outcome — not deep implementation details unless asked.

## Conduct

Be respectful. No harassment, spam, or bad-faith reports.
